# ToDo_Application
Created an api for ToDo application.

# Technologies Used:
Created this project using following technologies:

- Java
- Spring Boot
- Rest API
- MySQL
- Postman (For testing API's)

# Feature:

- Can perform all operations (GET, POST, PUT, PATCH, DELETE)
- Global Exception Handling
